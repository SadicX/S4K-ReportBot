# S4K ReportBot
Este programa escrito en python y de codigo abierto es un bot de reportes dedicado a videos y cuentas de instagram
Si deseas ver mas proyectos te invito a [mi GitHub] (https://github.com/SadicX)
Programa creado por SadicX y F11N.

## Setup

Dependiendo de su instalación de Python, sus comandos pueden variar.
A continuación se muestran los comandos necesarios para configurar.
El script ha sido probado en Python3, en Windows 10/11. Este script no se ha probado en ninguna otra plataforma.

Windows:
```
git clone https://github.com/SadicX/S4K-ReportBot
cd S4K-ReportBot
(Antes de usar el archivo python, debe usar el Isntalar Librerias.bat)
py ReportBot
```

## Uso
El Programa usará el metodo de reportes para intentar y mandar reports a una cuenta e intentar tumbarla
(No me hago cargo de cualquier uso que le de al programa o problemas)

## Errores

Si recibe un error:
Desinstale la escritura (python -m pip archivo) y vuelva a ejecutar el archivo py.
